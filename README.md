# Telegram Job Bot

> Разворачивается в Docker Compose на российском VPS.

Бот, который сам ищет IT-вакансии на пяти job-сайтах и присылает подходящие прямо в Telegram. Фильтры настраиваются кнопками в чате или в веб-панели — без правки кода.

## Как это работает

1. Вы создаёте фильтр: роль, город, зарплата, тип занятости, опыт, источники.
2. Раз в час бот опрашивает выбранные сайты по вашим ключевым словам.
3. Каждая новая вакансия проверяется по всем условиям фильтра и, если подходит, приходит карточкой со ссылкой.

Фильтры общие для Telegram и веб-панели — это одни и те же данные.

## Источники

| Сайт | Откуда берём данные |
|---|---|
| [hh.ru](https://hh.ru) | официальный API (с HTML-фолбэком) |
| [SuperJob](https://superjob.ru) | API (нужен бесплатный ключ) |
| [Работа России](https://trudvsem.ru) | Open Data API |
| [rabota.ru](https://rabota.ru) | парсинг HTML |
| [Хабр Карьера](https://career.habr.com) | парсинг HTML |

## Что умеет

- Пошаговый конструктор фильтра в Telegram — всё на кнопках.
- Точные фильтры: ключевые слова, слова-исключения, город, зарплата, тип занятости, опыт, набор сайтов.
- Строгое совпадение: выбрали «Удалённо» — вакансии другого формата (и без указанного формата) не покажутся. То же для опыта.
- Веб-панель на React: фильтры, результаты, история, избранное, блок-лист, статистика и здоровье источников.
- Очередь доставки в Telegram с автоповтором, если мессенджер временно недоступен.
- SQLite из коробки — отдельный сервер БД не нужен.

## Быстрый старт

```bash
git clone https://github.com/Lucky2356/telegram_bot_jobs.git
cd telegram_bot_jobs

pip install -r requirements.txt

cd web/frontend && npm install && npm run build && cd ../..

cp .env.example .env   # впишите свой BOT_TOKEN
python main.py
```

Бот, веб-сервер и планировщик запускаются вместе одной командой `python main.py`. На Windows есть `run.bat`.

### Минимальный `.env`

| Параметр | Обязательный | Где взять |
|---|---|---|
| `BOT_TOKEN` | да | [@BotFather](https://t.me/BotFather) → `/newbot` |
| `SUPERJOB_API_KEY` | нет | [api.superjob.ru](https://api.superjob.ru) — без него SuperJob пропускается |
| `WEB_PASSWORD` | нет | пароль для входа в веб-панель |

Остальные настройки (порт, интервал проверки, кэш, лимиты) — в `core/config.py`.

### Docker

```bash
docker compose up --build
```

Образ сам собирает фронтенд и поднимает бота вместе с веб-сервером на порту 8000.

## Telegram-команды

| Команда | Действие |
|---|---|
| `/start` | главное меню |
| `/add_filter` | создать фильтр (пошаговый мастер) |
| `/filters` | список фильтров |
| `/pause` · `/resume` | пауза и возобновление всех фильтров |

## Веб-панель

После запуска откройте [http://127.0.0.1:8000](http://127.0.0.1:8000) — там фильтры, результаты, история, избранное и ручной запуск проверки.

## Структура

```
main.py        точка входа: бот + веб + планировщик
core/          настройки, планировщик поиска, модели и БД
bot/           Telegram-бот на aiogram 3
scrapers/      парсеры пяти сайтов с общим интерфейсом
web/           FastAPI + React-панель (web/frontend)
alembic/       миграции БД
tests/         тесты (pytest)
```
